# Marine Bros

A simple platformer game inspired by Super Mario Bros, built with HTML5 Canvas and vanilla JavaScript.

## How to Play

1. Open `index.html` in your web browser
2. Use the arrow keys to move left and right
3. Press Space to jump
4. Jump on enemies to defeat them and earn points
5. Reach the golden star to win the game
6. Press R to restart at any time

## Game Features

- Classic platformer physics with gravity and jumping
- Multiple platforms to navigate
- Enemy characters that patrol back and forth
- Score tracking system
- Lives system (3 lives to start)
- Win condition (reach the golden star)
- Game over and restart functionality

## Controls

- **Arrow Left**: Move left
- **Arrow Right**: Move right
- **Space**: Jump
- **R**: Restart game

## Scoring

- Defeat an enemy: 100 points
- Reach the goal: 1000 points

## Files

- `index.html` - Main HTML structure
- `style.css` - Game styling
- `game.js` - Game logic and engine

Enjoy playing Marine Bros!
