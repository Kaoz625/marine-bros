# marine-bros — Project Overview
**Owner:** Kaoz625 | **Stack:** Vanilla JavaScript, HTML5 Canvas | **Status:** Playable MVP — missing levels, scoring, sound, and mobile controls

## What This Is
A browser-based side-scrolling action game built in pure vanilla JavaScript with no dependencies. Players control a marine character fighting through enemies in a side-scrolling environment. The game runs entirely in the browser with no server required.

## Current State
Core game loop exists in game.js with basic player, enemies, and rendering. Missing: multiple enemy types, level progression system, high score persistence, sprite animation improvements, sound effects, and mobile touch controls.

## Tech Stack
- Vanilla JavaScript (ES6)
- HTML5 Canvas
- CSS3
- No build tools — single file deployable
- Web Audio API (to be added)
- localStorage (to be added for scores)
